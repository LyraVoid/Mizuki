# Testing Custom URL Feature

You can completely customize article links through `url`.

With `url: "my-custom-path"` in the frontmatter, this post should be accessible at:
- `yoursite.example.com/my-custom-path/`

And you can still specify the url through `permalink: "my-custom-path"`. But it will only produce `yoursite.example.com/posts/my-custom-path/`.
